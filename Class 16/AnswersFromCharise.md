1. To repeat a block of code for a set amount of times ~~,or infinitely if you're bored and would like to watch your browser load forever~~.
  * For Loop:
      for(var i = 0; i < 10; i++) {
        window.print("https://goo.gl/bqTrZq")
      }

  * While Loop:
      var i = 0;
      while(true) {
        if(i === 10) {
          break;
        }
        window.print("https://goo.gl/bqTrZq")
        i++
      }
